# EnvironmentVarsApp

WPF приложение для управления переменными среды Windows с возможностью добавления комментариев.

## Технологии

- **.NET 8** - современная платформа разработки
- **WPF** - десктопный UI фреймворк
- **CommunityToolkit.Mvvm** - MVVM фреймворк
- **Serilog** - структурированное логирование
- **xUnit** - фреймворк для тестирования
- **FluentAssertions** - удобные утверждения в тестах

## Возможности

- 📖 Просмотр переменных среды из конфигурационного файла
- ✏️ Редактирование значений переменных
- 💬 Добавление и редактирование комментариев
- 🗑️ Удаление переменных
- 🔄 Обновление списка переменных
- 📝 Логирование всех операций

## Запуск

1. Клонировать репозиторий
2. Открыть `EnvironmentVarsApp.slnx` в Visual Studio
3. Собрать проект (Build)
4. Запустить `EnvironmentVarsApp` проект

## Конфигурация

Файл `appsettings.json` содержит список переменных для отображения:

```json
{
  "EnvironmentVariables": [
    "SMARTMEAL_API_KEY",
    "SMARTMEAL_SERVICE_URL",
    "SMARTMEAL_LOG_LEVEL",
    "SMARTMEAL_TIMEOUT"
  ]
}
```

## Требования

- Windows 10/11
- .NET 8.0 Runtime
- Права администратора для изменения системных переменных среды
